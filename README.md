# Game of Thrones Text Analysis using NLP
This project focuses on analyzing Game of Thrones dialogues using Natural Language Processing (NLP) techniques. The workflow includes text cleaning, preprocessing, and feature extraction using Bag of Words and TF-IDF to identify important words and patterns. Word2Vec is used to capture semantic relationships between words and understand contextual similarity. The project provides insights into dialogue structure, word usage, and emotional tone across the dataset, demonstrating the practical application of NLP techniques on real-world text data.
